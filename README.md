# Manitos Cool

Visualización 3D de manos usando puntillismo digital. Miles de partículas que forman manos rotando en el espacio.

## Lo Cool de las Matemáticas

- **Coordenadas esféricas** para distribuir puntos alrededor de los dedos
- **Cálculo de iluminación** con producto punto para las sombras
- **Transformaciones 3D** para las rotaciones suaves
- **Distribución probabilística** para crear volúmenes orgánicos

## Setup

```bash
npm install
npm run dev
```
